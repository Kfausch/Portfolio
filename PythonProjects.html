Python Applications

Password Generator

This is an application that can generate random passwords that contain numbers, symbols, upper and lower case letters. The Data is saved in a json file that can be returned.

Pong

This application is a Pong clone written in Python that utilizes the Turtle module. When the ball is created a random x,y coordinate is given to make each attempt unique. When a player scores the ball will freeze for a moment, the scoreboard will update, and then the ball will initialize again in the opposite direction.

Trivia

This is a trivia application that with a UI that runs on the tkinter module. New questions are generated from an Open Trivia Databse API on each attempt. When the player selects an answer the card will flash green when correct or red when incorrect.

Snake

This application is a Snake clone written in Python that utilizes the Turtle module. High scores are saved in a text file and only updated if the player exceeds the highest score. Each time the player reaches the food, the snake grows and the score is incremented by 1. If the snake collides with a wall or itself the game resets.
